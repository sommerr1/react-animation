# Мобильная версия - Изменения

## Выполненные изменения:

### 1. Удален верхний хидер для мобильной версии
- Добавлен CSS медиа-запрос `@media (max-width: 768px)` 
- Хидер с заголовком "3D Model Viewer" и подзаголовком скрыт на мобильных устройствах

### 2. Скрыты кнопки "нет" и "кивок" на мобильной версии
- Кнопки `.nod-btn` и `.shake-btn` скрыты через CSS на экранах меньше 768px
- Функциональность анимаций сохранена для десктопной версии

### 3. Добавлены жесты смахивания
- **Левое смахивание** - переход к следующей модели
- **Правое смахивание** - переход к следующей текстуре/цвету

### Технические детали:

#### Новые файлы:
- `src/hooks/useSwipeGestures.ts` - хук для обработки жестов смахивания
- `src/hooks/useTextureNavigation.ts` - хук для навигации по текстурам

#### Обновленные файлы:
- `src/store/modelSlice.ts` - добавлены действия для навигации по моделям и текстурам
- `src/components/ModelViewer.tsx` - интегрированы жесты смахивания
- `src/components/OBJViewer.tsx` - интегрированы жесты смахивания
- `src/App.tsx` - обновлен для использования Redux состояния
- `src/App.css` - добавлены стили для мобильной версии и подсказок

#### Функциональность:
- Смахивание работает только на мобильных устройствах
- Добавлены визуальные подсказки внизу экрана для мобильной версии
- Навигация по моделям и текстурам работает циклически
- Все изменения совместимы с существующей функциональностью

## Как использовать:
1. Откройте приложение на мобильном устройстве
2. Смахивайте влево для смены модели
3. Смахивайте вправо для смены текстуры/цвета
4. Подсказки отображаются внизу экрана
